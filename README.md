# Library Management System

## Project Overview
The **Library Management System** is a simple full-stack application designed to manage books in a library. It allows users to perform CRUD operations (Create, Read, Update, Delete) on book records efficiently. This project serves as an assignment to demonstrate full-stack development skills using modern technologies.

**Technologies Used:**
- **Backend:** C# .NET 7, ASP.NET Core Web API, Entity Framework Core, SQLite  
- **Frontend:** React.js, TypeScript, Axios, React Router  
- **API Documentation:** Swagger  

**Core Functionalities:**
- Add new books  
- View all books  
- Update existing books  
- Delete books  

---

## Features
- **Add New Book:** Users can create a book by specifying title, author, and description.  
- **View Books:** Displays a list of all books with details.  
- **Edit Book:** Modify existing book information.  
- **Delete Book:** Remove books from the library.  
- **Responsive Design:** Optimized for desktop and mobile devices.  
- **API Documentation:** Swagger UI for backend API testing.
